# 🇺🇦 DiiaClone — Java Edition

> Пародия на украинское государственное приложение «Дія» с Telegram-интерфейсом

---

## 📌 О проекте

**DiiaClone** — это учебный проект, который имитирует основную логику приложения «Дія»:  
хранение и управление цифровыми документами пользователя через личный кабинет.

📱 Взаимодействие с пользователем происходит через **Telegram-бота**,  
который общается с Java-бэкендом по REST API.

---

## 🏗 Архитектура
🤖 Telegram Bot  →  Spring Boot REST API  →  PostgreSQL
   (фронтенд)           (бэкенд)             (база данных)

---

## 🔐 Авторизация

В проекте используется **JWT (JSON Web Token)**:

- Пользователь логинится через бота
- Бот получает JWT-токен
- Токен передаётся в каждом запросе к API
- Сервер валидирует токен и даёт доступ к данным

---

## 📡 API Endpoints

### 🔑 Авторизация
- `POST /api/auth/register` — регистрация пользователя  
- `POST /api/auth/login` — вход и получение JWT  

---

### 📄 Документы
- `GET /api/documents` — получить все документы пользователя  
- `GET /api/documents/passport` — получить паспорт  
- `GET /api/documents/inn` — получить ИНН  
- `POST /api/documents/add` — добавить новый документ
